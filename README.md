# vue-vuex-router-boilerplate

## Features
- Vue + vue-router + vuex

## Directorys

public .... 本番用のファイル(minify後のJS,CSSや圧縮後の画像など)  
dest ...... 開発用のファイル(minifyされてないJS,CSSなど)  
src ....... 開発用のソースファイル  
devtools .. gulpやwebpackの設定ファイル  

## Build Setup

```bash
# モジュールのインストール
yarn install

# 開発
yarn dev

# 本番用のビルド
yarn build

# 本番の動作確認
yarn start
```
