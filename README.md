# vue-vuex-router-boilerplate

## Features
- Vue + vue-router + vuex

## Directorys

```bash
.   
├ public # 本番用ファイル(minify後のJS,CSSや圧縮後の画像など)  
├ dest # 開発用ファイル(minifyされてないJS,CSSなど)  
├ src # 開発用ソースファイル  
├ devtools # gulpやwebpack設定ファイル  
```

## Build Setup

```bash
# モジュールのインストール
$ yarn install

# 開発
$ yarn dev

# 本番用のビルド
$ yarn build

# 本番の動作確認
$ yarn start
```
